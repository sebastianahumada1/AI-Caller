## File Descriptions

### Configuration Files

- **package.json**: NPM package configuration with dependencies, scripts, and metadata
- **tsconfig.json**: TypeScript compiler configuration with strict typing and ESM
- **env.example**: Environment variable template with all required configuration

### Source Code

- **server.ts**: Main Express application with middleware, routes, and error handling
- **vapi.ts**: Vapi webhook handler with token validation and message type routing
- **ghl.ts**: GoHighLevel connector that dispatches tools to appropriate webhook URLs
- **schemas.ts**: Zod validation schemas for all request/response types and TypeScript interfaces

### Utilities

- **utils/logger.ts**: Structured logging with context support and request tracking
- **utils/http.ts**: HTTP client wrapper with timeout, retry logic, and error handling

### Generated Files

- **dist/**: TypeScript compilation output for production deployment
